# GangurDmitriy_TestTask
Тестовое задание для ПГК

В папке Pages интересующие нас Razor страницы Index(Отображение пользовательского интерфейса) и 
AdministrativePanel(Отображение, Аутентификация и обработка запросов административной панели). 
Папка Controllers, файл AjaxController.cs обработка Ajax запросов. Папка Models, файл Model.cs 
объект базы данных, объект монетного баланса и статический класс управления доступом.

При возврате сдачи показывает количество и номинал монет. 

База данных формируется из объекта Models.Beverage.
